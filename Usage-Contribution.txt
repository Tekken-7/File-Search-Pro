
File Search Pro
File Search Pro is a powerful and user-friendly file indexing and searching application designed for Windows.
It allows users to quickly index directories, search for files in real-time, filter results, and perform various file management tasks.
The application is equipped with features like tagging, emailing files, and support for both light and dark modes.

Usage
Indexing a Directory
Click Add Directory to select a directory to index.
Indexed files will appear in the results list with real-time monitoring.
Searching
Use the search bar to filter results by file name or keywords.
Filters
Select from Common Files Filter or Dev/Eng Files Filter in the dropdown menus to refine results.
Tagging
Right-click a file and choose Add/Edit Tags to organize files with tags.
Email Files
Right-click a file or use Ctrl+Click to select multiple files, then choose Send as Email... to attach them to an Outlook email.
Save Files
Right-click a file and select Save As... to copy it to a different location.
